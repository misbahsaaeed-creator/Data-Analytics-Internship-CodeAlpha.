Data Analytics Internship - CodeAlpha
This repository contains the tasks completed during my Data Analytics internship at CodeAlpha. The project focuses on Exploratory Data Analysis (EDA) and Data Visualization using the Titanic dataset.
Project Overview
The goal of this project was to analyze the Titanic dataset to identify factors that influenced the survival of passengers. I performed data cleaning, analyzed trends, and created impactful visualizations to tell the data story.
 Tasks Completed
Task 2: Exploratory Data Analysis (EDA)
Data Cleaning: Handled missing values in the 'Age' column using mean imputation and removed the 'Deck' column due to high missing data.

Data Structure: Explored the dataset using info(), describe(), and shape.

Pattern Detection: Identified key relationships between passenger demographics and survival rates.

Task 3: Data Visualization
Tools Used: Python, Pandas, Matplotlib, Seaborn.

Key Visuals:

Pie Chart: Overall survival percentage (38.4% survived).

Bar Charts: Showcased survival rates by Gender and Passenger Class.

Heatmap: Visualized correlations between numerical features.

📊 Key Insights
Gender: Females had a significantly higher survival rate than males.

Class: First-class passengers were more likely to survive compared to third-class passengers.
